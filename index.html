Recognizing and Translating Indian Sign Language (ISL) Hand Signs

This project aims to improve communication access for individuals with hearing impairments who use ISL, and promote their social inclusion and participation in society. Ultimately, this project aims to provide a reliable and accessible communication tool for individuals with hearing impairments who use ISL, allowing them to participate fully in society and improve their quality of life.

About Project

This project aims to develop an ML system that detects and recognizes Indian Sign Language (ISL) gestures, translating them into corresponding alphabets. The ML models used include ResNet50, VGG16, and EfficientNet-B3.

The gestures are recognized by these models and then translated into alphabets using a mapping between ISL gestures and alphabets. The translated text is displayed as output, facilitating communication between ISL users and English alphabet users.

The ML system developed offers an accurate and efficient solution for detecting and recognizing ISL gestures, enabling translation to alphabets. This project contributes to bridging the communication gap between ISL users and non-ISL users, promoting inclusivity and accessibility in society.
